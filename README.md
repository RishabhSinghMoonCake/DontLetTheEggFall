# 🥚 Don't Let The Egg Fall!

**A fun, casual, and delightfully wacky mobile game built around unpredictable gameplay and pure chaos.**

---

## 🎮 About The Game

**Don't Let The Egg Fall!** is a **lighthearted casual game** where things don’t always go as expected — and that’s exactly the point.

The objective is simple: *don’t let the eggs fall.*
But with quirky mechanics, unpredictable movements, and chaotic interactions, every moment turns into a fun and slightly absurd challenge.

This project focuses on creating a **playful and entertaining experience**, where randomness and weirdness make the game uniquely enjoyable.

---

## 🥚 Gameplay

* Eggs behave in **unexpected and sometimes ridiculous ways**
* You interact, react, and adapt to keep them from falling
* The challenge comes from **embracing the chaos** rather than controlling it
* Every run feels **different, unpredictable, and fun**
